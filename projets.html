Mes Projets Réalisés

Système de Détection de Cyberattaques avec IA

Technologies : Python, Machine Learning, Deep Learning, Django

Description :

Développement d'un système complet de détection de cyberattaques pour l'Industrie 5.0. Le projet inclut la capture de trafic réseau, le développement de modèles IA (XGBoost, Random Forest, LSTM, BPNN), et une interface web interactive pour la surveillance en temps réel.

Data Warehouse et Dashboard BI

Technologies : SSIS, SSAS, Power BI, SQL Server

Description :

Création d'un entrepôt de données et développement d'un tableau de bord Power BI pour l'analyse des ventes. Le projet comprend des processus ETL pour l'extraction et la transformation des données depuis AdventureWorks2012.

Site Web Personnel

Technologies : HTML5, CSS3, JavaScript

Description :

Développement d'un site web personnel moderne et responsive présentant mon parcours, mes compétences et mes projets. Le site inclut un quiz interactif et un formulaire de contact.
